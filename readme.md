## Pattern Punch Knit

Instructions to build

1. Install Brew. Go to https://brew.sh/ and follow instructions.

2. Use brew to install node and pnpm. https://formulae.brew.sh/formula/node

`brew install node`

`brew install pnpm`

3. Build the project.

`pnpm install`

`pnpm run build`

`pnpm run open`
